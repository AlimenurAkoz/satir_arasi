# 📚 Satır Arası - Kişisel Kitaplık ve Topluluk Platformu

**Satır Arası**, kitap severlerin kendi dijital kitaplıklarını yönetebildiği, profil bilgilerini özelleştirebildiği ve diğer kitapseverlerle etkileşim kurabildiği modern bir web platformudur. Bu proje, **İnternet Teknolojileri** dersi kapsamında modern web geliştirme standartları kullanılarak geliştirilmiştir.

---

## ✨ Özellikler

- **🔐 Dinamik Kimlik Doğrulama (Auth)**  
  Firebase Auth kullanarak kayıt olma ve giriş yapma sistemleri.

- **👤 Kullanıcı Profili Yönetimi**  
  - Kullanıcı adı güncelleme  
  - Profil fotoğrafı yükleme ve anlık önizleme  
  - Şifre değiştirme ve hesap silme işlemleri

- **🖼️ Görüntü Optimizasyonu**  
  Kullanıcıların yüklediği yüksek çözünürlüklü fotoğrafları **tarayıcı tarafında sıkıştıran** özel algoritma.

- **📬 İletişim Sistemi**  
  Firebase Firestore entegreli dinamik iletişim formu.

- **🎨 Modern Arayüz (UI)**  
  Glassmorphism (buzlu cam efekti) temalı, kullanıcı dostu ve **responsive** tasarım.

- **🧩 Ortak Bileşenler**  
  Tüm sayfalarda tutarlı çalışan merkezi **Navbar** ve **Footer** yapısı.
